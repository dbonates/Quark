ContentType
  • ContentType

    GRMustache distinguishes Text from HTML.

    Content type applies to both templates, and renderings:

    • In a HTML template, {{name}} tags escape Text renderings, but do not escape HTML renderings.

    • In a Text template, {{name}} tags do not escape anything.

    The content type of a template comes from Configuration.contentType or {{% CONTENT_TYPE:... }} pragma tags. See the documentation of Configuration.contentType for a full discussion.

    The content type of rendering is discussed with the Rendering type.
